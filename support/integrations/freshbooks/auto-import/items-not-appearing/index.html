When I create a client, project, or invoice in FreshBooks, it doesn’t appear in Cushion.

1. Check that the webhook exists.

The Auto-Import integration uses FreshBooks webhooks to tell Cushion when to create new clients, projects, and invoices. Upon enabling the integration, Cushion will automatically set up a webhook for you. To check that the webhook has been successfully created, go to /webhooks in your FreshBooks account (https://yourcompany.freshbooks.com/webhooks).

If the webhook wasn’t created, you should see a “No webhooks were found” message. Return to Cushion and try enabling the integration again.

2. Check that the webhook is verified.

FreshBooks webhooks need to be verified in order to start working. Cushion should handle this automatically, but in the off-chance the webhook wasn’t verified properly, you can verify it manually on the /webhooks page.

If the webhook hasn’t been verified, you should see something similar to the screenshot above. To try verifying it again, click the webhook’s checkbox, then click the “Resend Verification Code” button.

FreshBooks will reload the page and show the success message above, but still might not indicate that the webhook has been verified. Try waiting a few seconds, then refresh the page.

The webhook should now say “verified” in green. If not, try refreshing the page again.

3. Contact support.

If you’re still having issues, contact support and we’ll take a closer look.
