FreshBooks

Auto-Import

About this Integration

The FreshBooks Auto-Import integration will:

  • import clients, projects, and invoices shortly after you create them in FreshBooks
  • mark Cushion invoices as paid when payments are created in FreshBooks

Things to Know

  • This integration does not continue to update clients, projects, or invoices in Cushion once imported from FreshBooks.
  • If an imported FreshBooks project is flat-rate, its rate will be used as the estimated income in Cushion.

  • FreshBooks doesn’t link invoices to individual projects, so when a FreshBooks invoice is imported into Cushion, it will show (no project) under the project column. You must manually set the invoice’s project to maintain accuracy of the budget graph.

  • If a project or invoice is imported from FreshBooks and its client doesn’t exists in Cushion or hasn’t been linked to FreshBooks, the item will show (no client) under the client column. You will need to manually set the project or invoice’s client. That client will then be linked to FreshBooks, so any new projects or invoices imported from FreshBooks will automatically be linked to it.
